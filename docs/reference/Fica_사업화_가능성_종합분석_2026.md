# 피카(Fica) 사업화 가능성 종합 분석 보고서

> 기술적, 비즈니스적, 마케팅적, 규제적 관점 + 예비창업패키지 심사위원 비판적 평가

---

## 1부: 기술적 현실성

### 1-1. 경쟁사 현황 — "블루오션이 아니다"

| 제품 | 상태 | 기술 | 피카와 비교 |
|------|------|------|-----------|
| **Kemtai** (이스라엘, $5.8M 투자) | 운영 중 (B2B 전환) | 44포인트 추적 + 실시간 폼 피드백 | **직접 경쟁자** — 피카가 하려는 것을 이미 하고 있음 |
| **Peloton IQ** (2025.10 발표) | 출시 예정 | Movement-Tracking Camera, 폼교정+횟수카운팅+무게조절 제안 | **메가 경쟁자** — 자본력 비교 불가 |
| **VAY Sports** (스위스 ETH) | 운영 중 | 스마트폰 카메라 실시간 자세 코칭 | 직접 경쟁자 |
| **Kaia Health** ($285M 인수) | Sword Health에 합류 | CV 57포인트 추적, 임상 검증 완료 | B2B2C + 헬스케어 모델로 성공 |
| **Sculptor, Agit, Rep Tracker, GymCam** | 운영 중 | 카메라 rep 카운팅 앱들 | rep 카운팅은 이미 레드오션 |
| **Onyx** | **2022년 서비스 종료** | 카메라 AI 피트니스 | CV B2C 실패 사례 |
| **Mirror** (lululemon) | **2024년 단종** | $500M 인수 후 $443M 손실 | 하드웨어 모델 실패 증거 |

**핵심 발견**:
- "무게 자동 감지"까지 하는 앱은 **현재 없음** → 이것이 진짜 차별화 포인트
- 하지만 rep 카운팅 + 폼체크는 이미 다수 경쟁자 존재 → 사업계획서의 "경쟁자 없다" 주장은 **사실과 다름**
- CV 기반 B2C 피트니스 앱 중 **단독으로 수익을 낸 확인된 사례가 없음** (Onyx 종료, VAY 저활동, Kemtai B2B 전환)

### 1-2. MediaPipe 실전 한계

| 항목 | 연구실 | 실제 헬스장 | 비고 |
|------|--------|-----------|------|
| 정확도 | 97%+ | **70~90%** | 조명/가림/멀티유저로 10~20% 하락 |
| 벤치프레스 | 보통 | **매우 낮음** | 누운 자세에서 정확도 현저히 저하 |
| 배터리 | - | 시간당 15~25% | 30분 운동은 OK, 1시간은 도전적 |
| 멀티유저 | 1인만 | 혼동 발생 | 뒤에 사람 지나가면 추적 튐 |

- **88~95% 정확도 주장**: 서서 하는 운동(스쿼트, 데드리프트)에선 현실적. 벤치프레스 등은 70~85%
- Samsung Galaxy S23에서 추론 0.774ms → **성능은 문제없음**

### 1-3. YOLO 플레이트 인식 — "킬러피처이자 최대 리스크"

- "바벨에 플레이트가 있다" 탐지 → **가능** (90%+)
- "20kg 2장 + 10kg 1장 = 50kg" 정확 판독 → **매우 어려움**
- 같은 색(올블랙) 플레이트 구분, 겹친 플레이트 개수 세기 → 난제
- 200~500장 학습 시 mAP 65~80%, 15,000장으로 90~95% (SpaceO 사례)
- **현실적 전략**: "완전 자동"이 아닌 "AI 추측 → 사용자 1탭 확인" (반자동)

### 1-4. 폼체크 AI

- **Kaia Health 임상 검증**: CV 폼 교정이 물리치료사와 동등 정확도 (r=0.828) → 기술적 가치 입증
- **하지만**: 최고 성능 앱도 임상적 결함의 1/3을 놓침
- **법적 위험**: "AI가 안전하다고 했는데 부상" 소송은 시간문제
- **전략**: "교정"이 아닌 "참고용 피드백"으로 포지셔닝 필수

## 2부: 비즈니스/시장 현실성

### 2-1. 시장 데이터 (팩트)

- 글로벌 피트니스 앱 시장: 2025년 **$121억** → 2033년 $336억 (CAGR 13.4%)
- AI 피트니스 시장: 2024년 **$98억** → 2034년 $461억 (CAGR 16.8%)
- 한국 피트니스 앱 시장: 2024년 **$1.06억** → 2030년 $2.53억 (CAGR 15.4%)
- 68% 사용자가 "학습하고 적응하는" 플랫폼 선호 (McKinsey 2025)

### 2-2. 전환율 벤치마크 (핵심)

### 2-4. 성공/실패 사례에서 배운 것

**성공한 CV 피트니스**: Kaia Health ($285M 인수)
- 비결: B2C가 아닌 **B2B2C** (고용주/보험사 파트너십) + **임상 검증**

**실패한 CV B2C**: Onyx (종료), Mirror ($443M 손실)
- 원인: B2C 단독 수익화 실패, 고가 하드웨어 모델 한계

**생존 중**: Kemtai, Freeletics(5,900만 사용자)
- Kemtai: B2B 전환으로 생존
- Freeletics: AI 코칭 B2C로 €2,620만 매출 — 하지만 5,900만 다운로드에 도달하는 데 10년+

**한국**: 플랜핏 140만 다운로드, 글로벌 40% — 수익성 미확인

## 3부: 마케팅/전략 현실성

### 3-1. Jeremy Buendia 협업

| 강점 | 약점 |
|------|------|
| 인스타 400만 팔로워 | 보디빌딩 팬 ≠ 일반 피트니스 사용자 (타겟 미스매치) |
| 미국 시장 접근성 | 메가 인플루언서 전환율 1.7% (마이크로 6.3%의 1/4) |
| 브랜드 신뢰도 | 한국 시장 영향력 제한적 |

**현실적 판단**: "있으면 좋은" 수준이지 "게임 체인저"는 아님. 한국 피트니스 마이크로 인플루언서 10~20명이 더 효과적.

### 3-2. "미국 먼저" 전략

**비현실적.** 2인 팀으로 미국 시장 선공략은 물리적으로 불가능.

**대안**: 한국 검증 → 일본/동남아 → 영어권 (플랜핏이 이 경로로 글로벌 40% 달성)

### 3-3. 마케팅 예산

- 3,600만원(약 $25,000)으로 글로벌 론칭?
- 피트니스 앱 CPI $2~5 기준 → 5,000~12,500 다운로드가 한계
- BEP에 필요한 200,000 다운로드와 **10~40배 격차**

## 5부: 예비창업패키지 심사위원 비판적 평가

### 예상 점수: 60/100 (선정 경계선 65~70)

| 평가 항목 | 배점 | 점수 | 핵심 이유 |
|----------|------|------|----------|
| 문제인식 (P) | 30 | 21 | 문제는 실재하나, **직접 경쟁자 Kemtai/VAY/Peloton IQ 누락이 치명적** |
| 실현가능성 (S) | 30 | 17 | **가장 약한 영역.** CV/ML 핵심 인력 미확보, Phase 2~3 기술 난이도 과소평가 |
| 성장전략 (S) | 30 | 16 | 매출 추정 4~10배 과대, 미국 우선 전략 비현실적, BEP 경로 불투명 |
| 팀 구성 (T) | 10 | 6 | 도메인+앱개발은 우수, CV/ML 인력 미확보 |

### 치명적 약점 3가지

1. **직접 경쟁자 은폐**: Kemtai, VAY, Peloton IQ가 이미 존재하는데 "경쟁자 없다" 주장 → 심사위원 신뢰 훼손
2. **기술 논리적 비약**: Phase 1(SDK 호출) 완료 → Phase 2~3(커스텀 모델 학습) 가능? = "자전거 타니까 비행기도 조종 가능" 논리
3. **매출 추정 비현실성**: 업계 벤치마크 대입 시 4~10배 과대. BEP 20만 다운로드 필요 vs 마케팅 예산으로 1만 확보 가능

### 핵심 강점 3가지

1. **MVP가 실제로 존재** — 15개 화면 + CV Phase 1 완성. 아이디어만 있는 팀이 아님
2. **대표 + CTO 콤비가 탁월** — 16년 트레이너(도메인) × 삼성전자 AI(기술). 심사위원이 좋아하는 구성
3. **센서→CV 피벗 판단이 정확** — Mirror $500M→단종, Tonal 밸류 $1.6B→$550M 하락. 하드웨어 피벗은 시장 데이터로 뒷받침됨

### "이것만 보완하면 선정 가능" 5가지

1. **경쟁자 정직하게 재작성** — Kemtai/VAY/Peloton IQ 인정 + "우리만의 차이점(무게 자동 감지, 한국 헬스장 특화)" 강조
2. **로드맵 축소·집중** — 12개월 = "횟수 카운팅 + 폼체크 + 한국 베타 1,000명"만. 미국/B2B는 후속 라운드
3. **CV/ML 개발자 확보** — 최소 파트타임 참여 확약(LOI). "채용 예정"→"합류 확정"으로 변경
4. **베타 사용자 데이터 확보** — V2log를 10~50명에게 사용하게 하고 "NPS X점, 평균 세션 Y회" 숫자 만들기
5. **매출 추정 3개 시나리오** — 보수/기본/낙관 + 업계 벤치마크 명시적 인용

## 6부: 종합 결론 — "방향은 맞다, 주장이 과하다"

### 피카 사업의 현실적 위치

```
[기술] ████████░░ 80%  — Phase 1 완료, Phase 2~3이 핵심 도전
[시장] ███████░░░ 70%  — 시장은 성장 중, CV B2C 수익화 사례 부재
[팀]   ███████░░░ 70%  — 도메인+앱개발 우수, CV/ML 인력 공백
[전략] █████░░░░░ 50%  — 미국 우선/글로벌 동시는 비현실적
[재무] ████░░░░░░ 40%  — 매출 추정 과대, BEP 경로 불투명
```

### 최종 권고

**"피카의 방향성은 옳다. 하지만 사업계획서가 현실보다 3배 부풀려져 있다."**

1. **B2C 단독이 아닌 B2B2C 모델 병행** — Kaia Health 성공 경로 참고
2. **글로벌 동시가 아닌 한국 먼저 검증** — 플랜핏 경로 참고
3. **"무게 자동 감지"를 핵심 차별화로 내세우되, "반자동"으로 MVP** — 완전 자동은 비현실적
4. **마케팅 표현 순화** — "PT 대체"→"PT 보완", "자세 교정"→"운동 가이드", "부상 예방" 삭제
5. **예비창업패키지는 보완 후 충분히 선정 가능** — 아이템 자체는 심사위원이 "재미있다"고 느낄 주제
