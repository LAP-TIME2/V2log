## 촬영 방법

### 1. 바벨에 원판 끼우기

```
⚠️ 중요: 반드시 바벨 슬리브에 끼운 상태로 촬영!
   → 바닥에 원판만 놓고 찍으면 안 됨
   → 실제 사용 환경과 동일해야 함

✅ 좋은 예: 바벨 랙에 바벨 올려놓고, 원판 끼운 상태
✅ 좋은 예: 벤치프레스 랙에 바벨 + 원판
❌ 나쁜 예: 원판만 바닥에 세워서 촬영
❌ 나쁜 예: 원판을 손에 들고 촬영
```

### 2. 촬영 각도 (매 조합마다 5가지)

```
각 원판 조합(예: 20kg×2장)을 아래 5가지 각도로 촬영:

[각도 1] 정면 (0°) — 원판 얼굴이 보이는 각도
   → 폰을 원판 바로 정면에서
   → 숫자/색상이 잘 보임

[각도 2] 비스듬히 30° — 원판 측면이 살짝 보이기 시작
   → 정면에서 한 발짝 옆으로
   → 원판 테두리가 살짝 보임

[각도 3] 비스듬히 45° — 측면과 정면이 반반
   → 대각선에서 촬영
   → 겹친 두께가 보이기 시작

[각도 4] 비스듬히 60° — 측면이 더 많이 보임
   → 원판 옆에서 촬영
   → ⭐ 겹침 두께가 가장 잘 보이는 각도

[각도 5] 거의 측면 (75-80°) — 원판 옆면만 보임
   → 바벨 축 방향에서 촬영
   → 겹침 개수가 가장 잘 구분됨
```

**시각적 가이드:**
```
        📱 (각도1: 정면)
        |
        ↓
   ┌─────────┐
   │  20 kg  │ ← 원판 얼굴
   └─────────┘

📱 (각도3: 45°)
  ↘
   ┌─────────┐
   │  20 kg  │]  ← 두께 살짝 보임
   └─────────┘

                     📱 (각도5: 측면)
                      ↓
   ┌─────────┐
   │  20 kg  │]]  ← 겹침 두께 확실히 보임
   └─────────┘
```

### 3. 촬영 거리 (매 각도마다 2가지)

```
[거리 1] 가까이 (50-80cm) — 원판이 화면의 60-80% 차지
   → 원판 디테일, 테두리, 두께가 선명

[거리 2] 멀리 (1-2m) — 원판이 화면의 20-40% 차지
   → 실제 앱 사용 시 거리와 비슷
   → 바벨 전체가 보임
```

### 4. 카메라 설정

```
✅ 폰 기본 카메라 앱 사용
✅ 해상도: 최대 (4:3 비율 권장)
✅ HDR: 켜기 (밝기 차이 줄여줌)
✅ 플래시: 끄기 (헬스장 조명 사용)
❌ 줌: 사용 금지 (화질 떨어짐)
❌ 필터: 사용 금지
```

### 5. 배경/환경 다양성

```
같은 조합이라도 배경을 다르게:

[배경 1] 스쿼트 랙 — 바벨이 랙에 걸린 상태
[배경 2] 벤치프레스 — 바벨이 벤치 랙에 걸린 상태
[배경 3] 바닥 — 바벨이 바닥에 놓인 상태 (데드리프트 시작 자세)
[배경 4] 거울 앞 — 거울 반사 있는 환경 (헬스장 흔한 상황)

조명도 다르게:
[조명 1] 헬스장 형광등 아래 (밝음)
[조명 2] 프리웨이트 구역 (좀 어두울 수 있음)
```

## 촬영 순서 (헬스장에서 효율적으로)

### 라운드 1: 20kg 원판 (가장 중요) — 약 15분

```
Step 1: 바벨에 20kg 1장만 끼우기
   → 5가지 각도 × 2가지 거리 = 10장 촬영
   → 다른 배경 2곳에서 반복 = 20장

Step 2: 20kg 2장 겹쳐서 끼우기
   → 5가지 각도 × 2가지 거리 = 10장
   → 다른 배경 3곳에서 반복 = 30장
   → ⭐ 양쪽 다 촬영 (왼쪽에서, 오른쪽에서) = +20장
   → 목표: 50-80장

Step 3: 20kg 3장 겹쳐서 끼우기
   → 같은 방식 = 30-50장
```

### 라운드 2: 15kg 원판 — 약 10분

```
Step 4: 15kg 1장 → 10장 (기존 데이터 보충)
Step 5: 15kg 2장 → 30-50장
```

### 라운드 3: 10kg 원판 — 약 10분

```
Step 6: 10kg 1장 → 10장 (기존 데이터 보충)
Step 7: 10kg 2장 → 30-50장
```

### 라운드 4: 혼합 (여유 있으면) — 약 10분

```
Step 8: 20kg + 10kg 혼합 → 20-40장
Step 9: 20kg + 15kg 혼합 → 20-40장
Step 10: 15kg + 10kg 혼합 → 20장
```

### 라운드 5: 소형 (여유 있으면) — 약 5분

```
Step 11: 5kg 2장 → 20-30장
Step 12: 2.5kg 2장 → 10-20장
```

## 촬영할 때 주의사항

### ✅ 해야 하는 것

- [x] **바벨에 끼운 상태로 촬영** (원판만 따로 X)
- [x] **클립(칼라) 끼우기** — 실제 사용 환경과 동일하게
- [x] **양쪽 다 촬영** — 왼쪽에서도, 오른쪽에서도
- [x] **폰을 세로/가로 둘 다** — 가로 위주, 세로도 몇 장
- [x] **흔들리지 않게** — 손 떨림 없이 선명하게
- [x] **원판이 프레임 안에 완전히 들어오게** — 잘리면 안 됨

### ❌ 하면 안 되는 것

- [ ] 원판 없이 빈 바벨 촬영 (이건 필요 없음, 기존 데이터 충분)
- [ ] 너무 먼 거리에서 촬영 (원판이 화면의 10% 미만)
- [ ] 줌 사용 (디지털 줌 = 화질 저하)
- [ ] 다른 사람 포함 (프라이버시)
- [ ] 거울에 비친 원판을 메인으로 (실제 원판만)

### ⚠️ 흔한 실수

```
실수 1: 한쪽만 촬영
   → 바벨 양쪽(왼/오)에서 다 찍어야 함

실수 2: 비슷한 각도만 반복
   → 5가지 각도를 의식적으로 바꿔야 함
   → 특히 60°, 75° 측면 각도 잊기 쉬움

실수 3: 배경이 다 같음
   → 스쿼트 랙, 벤치, 바닥 최소 2-3곳에서

실수 4: 2장 겹침에서 틈이 벌어짐
   → 원판을 바벨 슬리브에 밀착시킨 후 촬영
   → 클립으로 고정하면 확실
```

## 촬영 완료 후 다음 단계

1. **사진을 PC로 복사** → 위 폴더 구조로 정리
2. **Roboflow에 업로드** → 기존 프로젝트(`v2log-weight-plates`)에 추가
3. **라벨링** → 새 클래스명(`20kg_x2`, `20kg_x3` 등)으로 bbox 라벨링
4. **기존 단일 원판 데이터 재라벨링** → `plate_20kg` → `20kg_x1`로 변경
5. **Dataset v3 생성** → Train/Val/Test 분리
6. **Colab에서 학습** → 기존 노트북에서 클래스 수만 변경
